<h4>THE MAZE PROJECT</h4>

## Installation
* prerequisites: cmake, c-compiler(preferably gcc)
* Clone this repo: `git clone "https://github.com/Mvoii/castlemania"`
* Compile game: `make build`
* run the game: `make run`
* after exiting: `make clean`
